\# Excel Diff ToolExcelファイル同士を比較し  \*\*追加 / 削除 / 変更\*\* を自動検出するVBAツールです。社員名簿・商品マスタ・売上データなど  \*\*どんなExcel表でも差分比較できます。\*\*\---\# 特徴\- ヘッダー自動検出\- 比較キー 1〜3項目\- 差分の色表示\- 強調表示（視認性重視）\---\# ツール画面※ここにスクリーンショットを追加予定\---\# Quick Example（まず動きを理解）このツールの動きを  サンプルデータで説明します。\## 比較元ファイル|社員番号|氏名|部署||---|---|---||1001|山田|営業||1002|佐藤|人事||1003|鈴木|総務|\## 比較先ファイル|社員番号|氏名|部署||---|---|---||1001|山田|営業||1002|佐藤|経理||1004|田中|営業|\## 比較キー社員番号\## 実行Excelツールを開き  「差分比較を実行」ボタンをクリックします。\---\# 出力結果\## 差分一覧|区分|社員番号|変更内容||---|---|---||変更|1002|部署 人事 → 経理||削除|1003|比較元から削除||追加|1004|比較先で新規追加|\## 区分の意味|区分|意味||---|---||追加|比較先にのみ存在||削除|比較元にのみ存在||変更|同一キーで値が変更|\---\# 出力シートツール実行後  以下のシートが作成されます。|シート|内容||---|---||操作|結果サマリー||差分一覧|追加 / 削除 / 変更||削除一覧|削除された行||要確認|キー空欄 / 重複||比較先データ（色付）|差分を色表示||比較先データ（強調）|視認性重視表示|\---\# このツールの特徴\## 汎用差分ツール多くの差分ツールは  社員名簿専用ですがこのツールは  \*\*どんなExcel表でも比較可能\*\*です。\---\## ヘッダー自動検出ヘッダーは  \*\*1行目固定ではありません。\*\*そのため\- CSV取込データ\- 社内帳票\- Excel管理表にも対応できます。\---\## 視認性重視比較結果は\- 色表示\- 強調表示\- フォント拡大で表示されます。これは  \*\*差分の見落としを防ぐための設計\*\*です。\---\# 対応データ例\## 社員名簿|社員番号|氏名|部署||---|---|---||1001|山田|営業|\## 商品マスタ|商品コード|商品名|価格||---|---|---||P001|ノートPC|120000|\## 売上データ|日付|商品コード|売上||---|---|---||2026/3/1|P001|30000|\---\# 使い方① Excelツールを開く  ② マクロを有効化  ③ \*\*差分比較を実行\*\* ボタンをクリック  ④ 比較ファイルを選択  \- 比較元ファイル  \- 比較先ファイル  ⑤ 比較キーを選択  1〜3項目選択できます。\---\# サンプルファイル`sample` フォルダに\- 社員名簿\- 商品マスタ\- 売上データのサンプルがあります。\---\# 必要環境\- Excel\- マクロ有効\---\# LicenseMIT License\---\# AuthorExcel VBA Tool\---⭐ \*\*Star\*\*このツールが役に立ったら  \*\*Star をお願いします。\*\*
