# Excel Diff ToolExcelファイル同士を比較し  追加 / 削除 / 変更 を自動検出するツールです。社員名簿・商品マスタ・売上データなど  どんなExcel表でも差分比較できます。---# ツール画面![ツール画面](https://raw.githubusercontent.com/tatex2000/excel-diff-tool/main/Tool1.png)---# 差分結果![差分結果](https://raw.githubusercontent.com/tatex2000/excel-diff-tool/main/tool2.png)---# 特徴・ヘッダー自動検出  ・比較キー 1〜3項目  ・差分の色表示  ・強調表示（視認性重視）---# 使い方① Excelツールを開く  ② マクロを有効化  ③ 「差分比較を実行」ボタンをクリック  ④ 比較元ファイル / 比較先ファイルを選択  ⑤ 比較キーを指定---# 必要環境Microsoft Excel  マクロ有効---# LicenseMIT License
